# App 端设计蓝图

> 从 `docs/wo-bot-app.md` 迁移 | 最后更新：2026-07-05
> 注意：此为远期规划，App 尚未开始开发。

## 定位

多平台原生 App（Android / iOS / 鸿蒙），是用户与机器人交互的主要入口。

## 核心能力

- 局域网发现机器人、连接机器人
- 查看机器人信息，操控机器人运动
- 机器人操控模式切换（手动、半自动、全自动）
- 查看机器人摄像头
- 机器人消息收件箱
- 快捷控制面板
- 小程序平台
- 机载软件管理
- SSH 终端

---

## 一、机器人操控模式

### 手动模式
用户直接操控机器人运动，若等待一段时间无操控，则自动切换至其他模式。

### 半自动模式
机器人根据任务或已建立的环境图样进行巡线运动，用户可实时监看。

### 全自动模式
机器人根据任务或环境图样进行自主运动。

---

## 二、信息展示界面

- **电池和充电状态**：电量百分比、充电状态
- **网络**：Wi-Fi 信号/SSID、蜂窝信号/运营商
- **运行负载**：CPU/内存/硬盘使用率
- **设备状态**：电池温度、预计续航、IP/蓝牙/GPS、运行时长
- **环境状态**：温湿度、光强、燃气检测（基于搭载传感器）
- **子系统状态**：音频/传感器/运动控制/电源/摄像头/GPS 等模块在线状态

---

## 三、快捷控制栏

类似手机控制中心，快速切换模式/音量：

| 功能 | 说明 |
|------|------|
| 寻找设备 | 让机器人发声定位 |
| 手电 | 云台 LED 开关 |
| 去充电 | 一键返回充电底座 |
| 控制模式 | 手动/半自动/全自动 |
| 省电模式 | 关闭部分功能、降低采样率 |
| 静音/音量 | 静音 + 滑动调节 |

---

## 四、遥控界面

横屏操控界面，类似王者荣耀：

- **麦轮驱动**：左摇杆平移 + 右摇杆转动
- **摄像头画面**：单摄/双摄 + 云台角度摇杆
- **小地图**：位置显示 / 周边感知（超声波等传感器渲染）
