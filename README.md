# wo-bot-wiki

wo-bot 项目全局方案、协议与规范文档仓库。

## 目录

```
wo-bot-wiki/
├── 方案/           ← 架构设计、技术方案（跨子项目）
├── 通讯协议/        ← WebSocket / WebRTC / mDNS 协议定义
├── 踩坑记录/        ← 开发过程中的经验记录
└── 项目规范/        ← 文档写作、代码规范、协作流程
```

## 工作流

1. **修改方案** → 新建分支 → 编辑 → 提 PR → Review → 合并
2. **关联需求** → 在 GitHub Issue 中引用方案文档链接
3. **子项目 wiki** → 各子项目自有 `docs/` 目录管理专属文档
