# wo-bot 项目总览

## 一、项目简介

wo-bot 是一个**机器人远程控制与智能交互平台**。用户通过浏览器或手机 App，在任何网络环境下连接和控制机器人，实现实时视频监控、语音喊话、云台操控、自主运动等功能。

| 项目 | 内容 |
|------|------|
| GitHub 主页 | https://github.com/al96169 |
| 作者 | Antonio Leung |
| 开源协议 | MIT |
| 包名前缀 | com.antonioleung.wobot |
| 硬件平台 | NVIDIA Jetson（Orin NX/AGX） + 小车底盘 + 摄像头 + 云台 + 麦克风/扬声器 |
| 目标场景 | 家庭安防巡检、远程看护、娱乐互动、教育展示 |

## 五、通信架构

### 当前（同局域网）

```
┌──────────────┐                              ┌──────────────┐
│  浏览器/App  │ ← WebSocket (命令+语音) →    │  机器人       │
│              │ ← WebRTC DataChannel (消息) → │  Jetson      │
│              │ ← WebRTC MediaStream (视频) ← │              │
└──────────────┘                              └──────────────┘
```

### 目标（跨网络，阶段 3 完成后）

```
┌────────┐      ┌──────────────┐      ┌────────┐
│ 浏览器  │←WSS→ │  信令服务器   │←WSS→ │ 机器人  │
│ (任意网) │      │  + coturn   │      │(任意网) │
└────────┘      └──────────────┘      └────────┘
       ↕ WebRTC P2P/Relay ↕
```

## 六、技术栈

| 层 | 技术 |
|----|------|
